# restaurant-europe
This code is for aggregation of kaggle's dataset about tripadvisor data of restaurants in Europe (https://www.kaggle.com/datasets/stefanoleone992/tripadvisor-european-restaurants). The result is used in my Tableau public dashboard (https://public.tableau.com/app/profile/aleksandr.tropin/viz/RestaurantEurope/DBRestaurantEurope).
